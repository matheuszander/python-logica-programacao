# CRUD de Usuários em Python

Este projeto implementa um sistema simples de CRUD (Criar, Buscar, Editar e Excluir)
usuários utilizando Python, listas e dicionários.

O objetivo do projeto é praticar lógica de programação e conceitos fundamentais
utilizados em backend, como funções com retorno, passagem de parâmetros e manipulação
de estruturas de dados.

## Funcionalidades

- Cadastrar usuário (nome e idade)
- Buscar usuário pelo nome
- Editar dados de um usuário existente
- Excluir usuário da lista
- Listagem controlada por menu

## Conceitos praticados

- Funções com `return`
- Tratamento de `None`
- Uso de listas e dicionários
- Busca por referência
- Edição e remoção de elementos da lista
- Separação entre lógica e interação com o usuário

## Tecnologias

- Python 3.x

## Observação

Projeto desenvolvido como parte dos meus estudos iniciais em lógica de programação
com foco em backend.
