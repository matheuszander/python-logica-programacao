# Menu de Números em Python

Projeto simples em Python criado para praticar lógica de programação, funções e manipulação de listas, sem uso de variável global.

## Funcionalidades
- Adicionar números a uma lista
- Mostrar os números cadastrados
- Somar todos os números da lista
- Limpar a lista
- Sair do programa

## Conceitos praticados
- Funções (`def`)
- Parâmetros
- Listas
- Estruturas de repetição (`while`, `for`)
- Condicionais (`if / elif / else`)
- Tratamento de erros (`try / except`)
- Separação entre lógica e controle (menu)

## Observações
Este projeto foi desenvolvido como parte dos estudos iniciais em Python, com foco em lógica de programação para o curso de Análise e Desenvolvimento de Sistemas (ADS).
