# Calculadora Simples em Python

Calculadora em modo texto com menu interativo.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão (com validação de divisão por zero)
- Menu de opções

## Conceitos praticados
- Funções com parâmetros e return
- Estruturas condicionais (if/elif)
- Laço while
- Tratamento de erros (try/except)
- Separação de lógica e interface

## Tecnologias
- Python 3.x
