# Calculadora Simples em Python

Projeto de uma calculadora em modo texto desenvolvida em Python, com menu interativo
e operações matemáticas básicas.

Este projeto foi utilizado para praticar lógica de programação, organização de código
e refatoração, evoluindo de uma versão inicial para uma implementação mais limpa
e reutilizável.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão (com validação de divisão por zero)
- Menu interativo
- Execução contínua até o usuário sair

## Conceitos praticados

- Funções com parâmetros
- Uso de `return`
- Laço `while`
- Estruturas condicionais (`if / elif`)
- Tratamento de erros com `try / except`
- Retorno de múltiplos valores em funções
- Refatoração para redução de código repetido

## Tecnologias

- Python 3.x

## Observação

Projeto desenvolvido como parte dos meus estudos iniciais em programação,
com foco em construção de base sólida antes do uso de frameworks.
